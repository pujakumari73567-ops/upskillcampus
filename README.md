# 📁 Smart File Organizer

**Internship Project Report** **Prepared by:** Pooja Kumari (Amity University, Noida - BCA 2024)

## 📌 Project Overview
The **Smart File Organizer** is a Python-based desktop application designed to automate the process of sorting and managing cluttered directories. It scans a selected folder, categorizes files based on their extensions, and moves them into systematically named sub-folders (e.g., Images, Documents, Videos). 

## ✨ Key Features
* **Automated Categorization:** Automatically identifies and sorts files into specific folders (Documents, Images, Videos, Audio, Archives, and Others).
* **Modern User Interface:** Built with `CustomTkinter` for a clean, dark-themed, and user-friendly interactive experience.
* **Standalone Software:** Packaged as a `.exe` executable file. It runs on Windows without requiring Python installation.
* **Intelligent Handling:** Safely handles unidentifiable extensions and skips pre-existing folders to avoid errors.

## 🛠️ Technologies Used
* **Programming Language:** Python 3
* **Core Modules:** `os`, `shutil`
* **GUI Framework:** `CustomTkinter`
* **Packaging Tool:** `PyInstaller`

## 🚀 How to Use (Run the App)
1. Download the repository to your computer.
2. Go to the `dist` folder and double-click on `organizer.exe`.
3. Click on the **"Browse Folder"** button to select your target, messy directory.
4. Click **"Organize Now!"** and let the software do the magic!
